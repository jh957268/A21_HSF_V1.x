Intro

Hello and welcome to the new generation of wireless lighting control! This page will help you get the most out of your AKS, AKS Plus, or Satellite device. To start off, here are some quick tips:

-To start transmitting to unlinked Cintennas or CRMX fixtures, click the LINK button on your device or on the web interface.

-To unlink all linked Cintennas or CRMX fixtures in range, hold RF LINK for 5 seconds or click “Unlink” on the web interface.

-The DMX light will blink with every DMX frame received. Use it to indicate signal health!

-If Art-Net or sACN  is being sent, but no DMX signal appears, check your settings to ensure correct protocol and universe. Some consoles report Art-Net universe numbers differently! 

-There are three modes. To switch, simply click power 2x for WiFi Client, 3x for WiFi Host, and 5x for Ethernet Client.
-All settings can be reverted by powering on while holding RF LINK. DMX will blink when done.

BUTTONS and LEDS

 

RF LINK Button

-The RF LINK Button is used to Link and Unlink the CRMX transmitter to its receivers. It is found in the upper right-hand corner of the Satellite.

To Link: When unit is on, click the RF Link button to start the automatic linking process. Any unlinked crmx device in range will connect.

To Unlink: Press and hold RF Link button. All linked units in range will receive the Unlink signal and disconnect.

CRMX Status Indicator

This LED indicates CRMX Transmitter operation. It is solid when in transmit mode, and blinks when linking.

DMX Indicator

-The DMX LED indicates that DMX data is being received by the unit and transmitted via CRMX.

 

WIFI/Network Status Indicator

-The WiFi LED indicates the mode and operational status of the network portion of the device. It will be solid in AP mode, blink when in WiFi Client mode, and “Anti-Blink” when in Ethernet Client mode. The blink will become a double-blink when the unit is linked to a host.

 

BATT STAT/Charging Indicator

This LED indicates that the unit is plugged in and charging. It can also indicate a possible battery error.

LO BATT LED

-This LED indicates that the battery level is low. It blinks quicker as battery level drops.

POWER Button

-The POWER Button is used to power on the unit as well as change its mode. It is found in on the center face of the Satellite.

 

Power On/Off - 

     Power on the device by holding Power button for a few seconds when off.

     Power off AKS by holding power button for a few seconds when on.

 

Enable Access Point (AP) mode:

     Triple-click the Power button. WiFi light is solid on this mode. Other devices including iPads, WiFi enabled fixtures, and other AKS or Satellite units can connect to an AKS in this mode. Devices connected via Ethernet will share the network with WiFi. DHCP is enabled in AP mode by default.

Enable WiFi Client (STA) mode:

     Double-click the Power button. Power Indicator will Blink to indicate serial command (AT) activity. When process is complete, WiFi light will begin blinking as described below. Device will seek out host Access Point with Ratpac AKS SSID. The Ethernet port exists behind a gateway in this mode, and the unit can be accessed at 10.10.99.254

 

Enable Ethernet Client mode:

     Five-click the Power button. Power light will Blink as configuration occurs, which may take a minute. WiFi light will illuminate with occasional “anti-blink” when ready. WiFi will be disabled in this mode, and device will connect via Ethernet port to suitable hosts such as an AKS or router with DHCP server capability.

Web Interface

The Web interface can be accessed by entering the device’s IP address into a browser. In Host/AP mode, the default IP address is 10.10.100.254. In WiFi Client and Eth Client mode, the unit is dynamically assigned an IP address. When accessing a unit in WiFi Client mode directly via the local Ethernet port, the unit can be accessed at 10.10.99.254. 

 

Main Settings

The Main Settings Window will give you everything you need to configure a single AKS or Satellite unit. 



Mode - This is network mode the unit is currently in. It can be Host, WiFi Client, or Eth Client.

IP - This is the IP of the unit on its main network. 

Battery Level - This shows the battery level of the unit.

 

ID - This button causes the unit to blink its LEDs and identify itself. 

Link/Unlink - These buttons will Link or Unlink the CRMX transmitter remotely.

 

Art-Net/sACN Settings

The AKS and Satellite now support sACN as well as Art-Net.
Node Name: this name will be displayed in Art Poll Replies as well as Constellation.

Art-Net Subnet and Universe: Sets Art-Net Universe. Must be Net 0.

sACN Universe: Sets sACN Universe and Multicast Subscription address. Requires Restart. 

Protocol: Choose between Art-Net and sACN. Note: Changing requires Restart. 

Press “Apply” to save settings before moving on.

 

Output Power and Color ID

CRMX Power Level - Sets the output power of the CRMX Transmitter. 

Color ID: On Satellite, this will set the LED Color. All compatible receivers will reflect this value.

Gaffer’s Window
The Gaffer’s Window allows a second input source to control a range of channels on a single universe. To use, click “Enable”, and select the range of channels to be affected as well as the universe assignment of the second input source (must be different from main Art Net input universe). The signals will automatically be mixed.  On the AKS, the ability to optionally use the XLR input as either the primary or windowed universe can be enabled through the checkboxes.

AP (Host) Interface Settings

Network Mode - allows 802.11b/g/n or combinations of these standards to be selected. This setting can be used to increase compatibility with older tablets. 

Default: “11b/g/n mixed mode”

Network Name (SSID) - the WiFi Network name. It can be optionally hidden.
Default “Ratpac AKS”

Frequency (Channel) - the WiFi Access Point channel. To reduce interference, it is recommended that only 1, 6, or 11 are chosen.
Default: “1”
Caution: “AutoSelect” disables the CRMX Keep-Out Window!

Wireless Distribution System (WDS) - an optional feature set for infrastructural networks. This feature is not officially supported in the AKS, though it has been left functional for those who need it.
 

Security Mode - selects the WiFi encryption method. It is recommended to use the highest one supported by connecting devices.
Default: “WPA2-PSK”

WPA Algorithms - set which encryption algorithm shall be used. AES is recommended, but TKIPAES will allow devices to choose whichever they are compatible with.
Default: “TKIPAES”

Pass Phrase - the WiFi Password. 

Default: “quietonset”

IP Address - refers to the address in Host mode as well as Art-Net Node address.
Default: “10.10.100.254”

Subnet Mask - allows or filters ranges of IP addresses for the sake of network efficiency.
Default: “255.255.255.0” or “255.0.0.0”

 

Server address - should be the first three entries of the IP address with the fourth as “255”
This allows Art-Poll replies to be sent to all devices on the network in range.
Default: “10.10.100.255”

STA (WiFi Client) Interface Setting

To enable Station Mode, double-click the Power Button. Unit will automatically link to the nearest WiFi router with the name “Ratpac AKS” and password “quietonset”

 

AP’s SSID, which represents the host network, can be configured by pressing the “Search” button. A list of nearby networks will appear. If the desired network does not show up, click “Refresh.” This process may need to be repeated. When the desired network is found, select it and hit “Apply.”  
Enter the Passphrase, and press “Apply.”
Default SSID: “Ratpac AKS”
Default Pass: “quietonset”

Constellation

Constellation is the new, easy method to connect many AKS or Satellite units to a single network and configure multiple units at once. After connecting all clients to a host via WiFi or Ethernet switch, the list will automatically populate showing each unit.

Node Name - Art Poll Name of the client unit, also a link to its main page

IP Address - Address of the client unit

Battery - Battery level of the client unit

Identify - Causes client unit to blink for identification purposes

Link - Links Wireless CRMX Receivers to client unit

Unlink - Sends Unlink Command - linked receivers must be in range to be unlinked

Art-Net Sub/Universe - Client unit’s Art-net Universe. Click “Set” to change.

“Refresh - Refreshes the list of clients

Back to Top

Firmware Update Mode

 

In addition to updating firmware via the Device Manager page, firmware can also be updated for several units at once using the Firmware Update Mode. This mode shows all units that are present in constellation and options to update each of their firmwares.

Node Name - Art Poll Name of the unit, also a link to its main page

IP Address - Address of the unit

Battery - Battery level of the unit. Ensure this is at least 50% before updating!

Identify - Causes unit to blink for identification purposes

eCos Ver. - Current WiFi firmware version. “Update” loads file below.

SAMD Ver. -Current Processor firmware version. “Update” loads file below.

Refresh - Refreshes the list of clients

 

eCos Firmware Location: Choose the WiFi Firmware File to upload.

SAMD Firmware Location: Choose the Processor Firmware File to upload. 

 

Devices in list can be updated by clicking “Update” after choosing the file. A status message will appear below the list.

Factory Reset:

 

To reset all settings to factory default and revert to AP mode, press and hold both Power and RF Link buttons when unit is off. The device will power on and automatically reconfigure itself. The DMX light will Blink when done.
